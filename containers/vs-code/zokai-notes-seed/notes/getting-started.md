# Getting Started with Zokai Notes

> **New here?** Complete the [[setup-guide]] first to configure your API key and enable codebase indexing.

## Creating Notes

1. Create a new `.md` file in the `notes/` folder
2. Use `[[double brackets]]` to link to other notes
3. If the linked note doesn't exist yet, `Ctrl+Click` it to create it

## Wiki Links

Wiki links create connections between your notes:

- `[[note-name]]` — links to `note-name.md`
- `[[note-name|Display Text]]` — links with custom display text

## Finding Your Way

- **Backlinks Panel**: In the sidebar, see all notes that link TO the current note
- **Graph View**: `Ctrl+Shift+P` → "Show Graph"
- **Connections Panel**: See direct connections from the current note

## Using Templates

Check `notes/templates/` for reusable note templates:
- **Meeting Notes** — for capturing meeting outcomes
- **Spike Report** — for technical investigations
- **Daily Log** — for daily standups and progress

Copy any template into `notes/` and start editing.
